# USDZ Contract

Add one USDZ file per dish using the exact filenames referenced in `src/lib/menu/catalog.ts`.

Required filenames:

- `garden-citrus-salad.usdz`
- `stone-oven-pizza.usdz`
- `roasted-chicken-platter.usdz`
- `oven-grilled-chicken.usdz`
- `smokehouse-burger.usdz`
- `crispy-burger-fries.usdz`
- `creamy-herb-spaghetti.usdz`
- `momo-basket.usdz`
- `caramel-dessert-cup.usdz`

After each file is added, set the matching `usdzReady` flag to `true` in the catalog so the Quick Look CTA becomes active on iPhone/iPad.
